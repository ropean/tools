zip命令参数及释义

  • zip -r <压缩文件名.zip> <目标文件或目录> : 将指定的文件或目录压缩为一个zip文件（递归压缩子目录）。
  • zip -u <压缩文件名.zip> <文件> : 更新已存在的zip文件，添加或更新指定的文件。
  • zip -d <压缩文件名.zip> <文件> : 从zip文件中删除指定的文件。
  • zip -l <压缩文件名.zip> : 列出zip文件中的文件列表，显示文件名、压缩比率等信息。
  • zip -e <压缩文件名.zip> <文件> : 使用密码加密压缩文件，需要输入密码来解压文件。
  • zip -x <压缩文件名.zip> <排除文件或目录> : 创建zip文件时排除指定的文件或目录。
  • zip -q <压缩文件名.zip> <文件或目录> : 安静模式，不显示压缩进程的输出。
  • zip -m <压缩文件名.zip> <文件或目录> : 将指定的文件或目录添加到zip文件后，然后删除原文件。
  • zip -T <压缩文件名.zip> : 测试zip文件的完整性。
  • zip -c <压缩文件名.zip> <文件列表> : 创建一个只包含指定文件的zip文件。
  • zip -rj <压缩文件名.zip> <目标文件或目录> : 将指定的文件或目录压缩为一个zip文件，不包含目录结构。
  • zip -0 <压缩文件名.zip> <文件列表> : 使用存储模式压缩，不进行压缩，仅存储文件。
  • zip -v <压缩文件名.zip> : 显示压缩过程的详细输出。
  • zip -F <压缩文件名.zip> : 修复损坏的zip文件。
  • zip -r9 <压缩文件名.zip> <目标文件或目录> : 最大压缩级别，压缩比最高。
